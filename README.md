# Luxaltic.github.io
Luxaltic
Developed by Maxim Prosser

This program is a 2D configurator for companies to show different versions of their products.
The Admin can create new versions of products using an image of an existing product.
Different versions will vary by color and/or texture.
Each product can be viewed seperately in the gallery to compare the different versions.

How to Use:
Load the website
Make an account/sign-in/choose which company to view items of
View the gallery of items
Select item of interest
Select different styles to compare
